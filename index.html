Works

Password Generator

A random password generator that showcases primary Javascript skills. It consists of a starter code that is modified to create an application that enables employees to generate random passwords based on a criteria they have selected. The application runs in a browser and features a dynamically updated HTML and CSS, powered by Javascript.

Workday Scheduler

A Third-Party API calendar application or workday scheduler with a modified starter code that allows a user to save events for each hour of the day. The application runs in the browser and features dynamically updated HTML and CSS, powered by jQuery. The Moment.js library is used to work with date and time.

Notabilis (Note Taker)

Notabilis, which means noteworthy in Latin, is a note taker application that can be used to write, save, and delete notes. It uses an Express.js back end, and saves and retrieves note data from the JSON file.

Team Profile Generator

A Node.js command-line application that takes in information about employees in a software engineering team, then generates a HTML web page that displays summaries for each person.

README Generator

A command-line application that dynamically generates a professional README.md file from a user's input using the Inquirer package.

Code Quiz Challenge

A Web API assignment that features a timed, interactive coding quiz with multiple choice questions, for the purpose of familiarizing Bootcamp students with coding assessments that include both multiple choice questions and interactive coding challenges, as a part of the interview process for full-stack developers. The application runs in the browser and features dynamically updated HTML and CSS, powered by JavaScript.

Projects

Project 1: Food For Thought

An interactive web application that allows users to search for recipes with a single word or name of ingredient. As a member of the team, I worked on the application's wireframe using Google Slides and mock-up using Figma, as well as the HTML and CSS files. I was also responsible for integrating Bulma into the HTML and CSS framework, and the application's responsiveness when viewed in different devices. Last, I wrote the application's README, User Story, and Acceptance Criteria, and prepared screenshots of its functionalities for documentation.

Project 2: Oh My, Books!

This is an interactive web application that allows users to create and post reviews of books by creating a login account. Books that can be reviewed are found in the application's database and can be searched by title, genre, International Standard Book Number (ISBN), and author. As a member of the team, I have worked on the database, MySQL and JSON files, and the README. I have also captured images of the application's features and functionalities for documentation.

About Me
